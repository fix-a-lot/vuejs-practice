# vuejs-testbad

Vuejs 샘플 코드 저장소 겸 실험실

## 서버 실행

Node.js [express](https://expressjs.com) 패키지 사용함.

```bash
# 의존성 패키지 설치
npm install
# 혹은
# yarn install

# 서버 시작
node server.js

# nodemon으로 서버 시작
npm exec nodemon server.js
# 혹은
npx nodemon server.js
```

브라우저로 [http://localhost:8888](http://localhost:8888) 열기
